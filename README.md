## **Start FastAPI Server**  
repo: https://github.com/5mbl/10k-v2-API
```bash
cd backend  
source venv/bin/activate  
uvicorn main:app --host 0.0.0.0 --port 8000 --reload  
```
## **Start Frontend**  
```bash
cd frontend  
npm run dev    
```
